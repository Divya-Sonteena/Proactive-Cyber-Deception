About Cowrie

Cowrie is a medium-interaction SSH and Telnet honeypot. It emulates an SSH server and logs all commands entered by attackers, as well as credentials used in brute-force attacks. Data is tokenised into canonical sequences for ML analysis.

Captures: SCAN, RECON, LOGIN_ATT, LOGIN_OK, EXEC, FILE_XFER, TUNNEL, SESS_END

About Dionaea

Dionaea is a low-interaction honeypot that catches malware by emulating vulnerable services (SMB, HTTP, FTP, MSSQL, MySQL). It collects malware binaries and network exploits for analysis.

Captures: SCAN, EXPLOITATION, MALWARE, FILE_XFER, SESS_END
